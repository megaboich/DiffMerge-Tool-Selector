DiffMerge-Tool-Selector by megaboich

DiffMergeSelector is a simple application to help select diff tool on the fly.

You can call it "diff tool proxy". The main idea that you configure your IDE, SVN, Git, Mercurial, or whatever tool to use DiffMergeSelector as a merge tool. After that each time when DiffMergeSelector runs it easily allows to select the most appropriate Diff Tool.

Includes simple UI to configure all tools, also configuration stored in XML file and can be edited manually.
